Puzzle of the Farmer, the Wolf, the Goat, and the Cabbage

As usual, answer all questions in a plain text file named questions4.txt. At the top of your plain text file, please include your name, user id, and lab section. Failure to do so will result in a 0.

This classic problem requires that the farmer carry his wolf, goat, and cabbage across a river in a leaky boat. Because the boat leaks, the farmer can only take one of the three items in each trip across the river. However, if the wolf is left alone with the goat, it eats the goat. Similarly, if the goat is left alone with the cabbage, it eats the cabbage.

In this lab, you will create a digital logic circuit that will allow a user to attempt to solve this puzzle. You'll also publish this puzzle on your web page.

During lecture, we said we would be doing a number related lab this week. However, we had a slight change of heart and will be doing this one instead.

Question 1 (10pts)

Start by opening the logic workbench and wiring up the inputs as illustrated here.

The four toggle switches each represent the four things to move across the river: the farmer, the wolf, the goat, and the cabbage. When the corresponding switch is turned off, it represents the item being on the left side of the river. When it's turned on, it represents the item on the right side of the river. If your browser supports it, I recommend double clicking on the names of the switches and changing them to names that describe the items they represent. The push on switch is used to test a configuration to see whether it is a final solution, a safe state, or a dangerous state. Your first task for this lab is to create a circuit for detecting the successful completion of the task. You should design and construct a circuit that illuminates an LED when all four are on the right side of the river. Question 2 (15pts)

For using logic to express the behavior of this puzzle, we will have four Boolean variables: f, w, g, and c. These correspond to the farmer, the wolf, the goat, and the cabbage, respectively. For each of the variables, a True value indicates that the corresponding thing is on the right bank of the river, and a False value indicates that it is on the left bank.

The next step in developing this puzzle web page is to write a Boolean expression that is True when there is a potential problem on the right bank: that is when the farmer is not there and either the wolf and goat or the goat and cabbage are there. Remember, it's only a problem if the farmer is not there to keep control. Write down the truth table for the answer.
